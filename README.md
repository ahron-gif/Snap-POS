# Snap-POS — Web-Based Back Office

## Overview
Snap-POS is a comprehensive Point of Sale system for retail stores. This repository contains the **web-based Back Office** application, built with:

- **Frontend:** React.js
- **Backend:** .NET Core
- **Database:** Microsoft Azure SQL (connected to existing customer data)

## Architecture
- `/client` — React.js frontend application
- `/server` — .NET Core backend API
- `/docs` — Documentation and feature specifications

## Status
🚧 Under active development — migrating features from the desktop VB.NET/DevExpress application to the web platform.

## Customers
Currently serving ~60 retail store customers with the desktop version. This web-based Back Office will provide the same functionality through a modern browser interface.
